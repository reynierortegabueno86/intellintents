- request_recommendation: User asks for a personalized investment recommendation.
- request_portfolio_suggestion: User wants a suggested portfolio allocation or model portfolio.

- request_alternative: User wants different options or alternatives to the current recommendation.

- reject_recommendation: User disagrees or does not want to follow the recommendation.
- request_human_advisor: User wants to speak with a human financial advisor about investment advice.
- request_rebalance_advice: User asks for guidance on rebalancing their existing portfolio.
